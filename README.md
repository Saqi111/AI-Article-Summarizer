# AI Article Summarizer 

A simple web app that summarizes long articles into short, easy-to-read content. This project is built using Python, Gradio, and Hugging Face Transformers.

## Features
- Summarizes long English articles.
- Simple and clean user interface.
- Built with modern AI tools.

## Live Demo
You can try the live application here:
[Click here to see the Live App!]https://huggingface.co/spaces/Saqlaain/My-Summarizer-App

## Technology Stack
- **Backend & AI Logic:** Python
- **AI Model:** Hugging Face Transformers (`distilbart-cnn-12-6`)
- **Web Framework:** Gradio
- **Hosting:** Hugging Face Spaces
